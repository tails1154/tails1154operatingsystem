# Contributing

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Run `./build.sh` to verify the ISO builds
5. Submit a pull request

## Development

- Edit `packages.x86_64` to add/remove packages
- Edit `airootfs/` to customize the live environment
- Edit `profiledef.sh` for ISO metadata
- Calamares config is in `airootfs/etc/calamares/`
